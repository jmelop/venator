## Requirements

- **Node.js** >= 18
- **React** 18

Venator is framework-agnostic but is optimised for use with Next.js 14+ (App Router). All examples in this documentation use Next.js.

## Install packages

Install all three packages from your workspace or registry:

<CodeBlock
  language="bash"
  filename="Terminal"
  code={`npm install @venator/ui @venator/patterns @venator/tokens`}
/>

## Configure Tailwind

Add the package source paths to your `tailwind.config.js` content array so Tailwind can scan component class names. Also enable `darkMode: 'class'` for dark mode support.

<CodeBlock
  language="js"
  filename="tailwind.config.js"
  code={`/** @type {import('tailwindcss').Config} */
module.exports = {
  darkMode: 'class',
  content: [
    './src/**/*.{ts,tsx,md,mdx}',
    '../../packages/ui/src/**/*.{ts,tsx}',
    '../../packages/patterns/src/**/*.{ts,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};`}
/>

## Add ToastProvider

Wrap your app root with `ToastProvider` so that `useToast` is available anywhere in the tree.

<CodeBlock
  language="tsx"
  filename="app/layout.tsx"
  code={`import { ToastProvider } from '@venator/ui';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <ToastProvider>
          {children}
        </ToastProvider>
      </body>
    </html>
  );
}`}
/>

## First component

You're ready. Import any component from `@venator/ui` or `@venator/patterns` and start building.

<CodeBlock
  language="tsx"
  filename="page.tsx"
  code={`import { Button, Card, CardContent, CardHeader } from '@venator/ui';

export default function Page() {
  return (
    <Card>
      <CardHeader
        title="Welcome to Venator"
        description="Start building your interface."
      />
      <CardContent className="mt-4">
        <Button variant="primary">Get started</Button>
      </CardContent>
    </Card>
  );
}`}
/>
